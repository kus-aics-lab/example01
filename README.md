# Gradient Web GUI

`f(x,y)=x^2e^y`의 그래프와 그래디언트 방향을 웹에서 시각화하는 강의용 예제입니다.

## 실행 방법

1. `index.html` 파일을 브라우저에서 엽니다.
2. 인터넷이 연결되어 있으면 Plotly CDN이 자동으로 로드됩니다.

## 현재 기능

- 3차원 곡면 그래프 표시
- 등고선 그래프 표시
- 선택한 점에서의 그래디언트 방향 표시
- 임의 방향 `u` 표시
- 방향도함수 `D_u f = grad f · u` 계산
- `u`를 그래디언트 방향으로 맞추는 버튼
- `u` 자동 회전 애니메이션
- 등고선 그래프에서 클릭으로 점 선택
- 선택한 점 근처에서 드래그하여 방향 `u` 직접 설정

## 수업에서 보여주기 좋은 흐름

1. 등고선 그래프에서 임의의 점을 클릭합니다.
2. 그 점 근처를 드래그해서 방향 `u`를 바꿉니다.
3. 정보 패널에서 `D_u f = |grad f| cos(theta)`를 확인합니다.
4. `u 를 그래디언트 방향으로 맞추기` 버튼을 눌러 최대 방향도함수를 비교합니다.
5. `u 자동 회전`을 켜서 방향도함수가 어떻게 변하는지 애니메이션으로 보여줍니다.
