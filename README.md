# Gradient Web GUI

임의의 이변수 함수 `f(x,y)`를 입력하면 다음 내용을 웹에서 시각화하는 강의용 예제입니다.

- 3차원 곡면 그래프
- 등고선 그래프
- 선택한 점에서의 그래디언트 방향
- 임의 방향 `u`
- 방향도함수 `D_u f = grad f · u`

## 실행 방법

1. `index.html` 파일을 브라우저에서 엽니다.
2. 함수 입력창에 식을 넣고 `함수 적용`을 누릅니다.

## 입력 예시

- `x^2*exp(y)`
- `sin(x*y)`
- `x^2+y^2`
- `log(x^2+y^2+1)`

## 사용 방법

1. 함수식을 입력합니다.
2. 등고선 그래프에서 클릭하여 점을 선택합니다.
3. 선택한 점 근처에서 드래그하여 방향 `u`를 정합니다.
4. 정보 패널에서 `D_u f = |grad f| cos(theta)`를 확인합니다.
5. `u 를 그래디언트 방향으로 맞추기` 버튼으로 최대 방향도함수를 비교합니다.
6. `u 자동 회전`을 켜서 방향도함수 변화 애니메이션을 볼 수 있습니다.

## 참고

- 그래디언트는 수치미분으로 계산합니다.
- 함수가 어떤 점에서 정의되지 않으면 해당 영역은 빈칸처럼 보일 수 있습니다.
